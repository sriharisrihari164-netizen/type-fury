# Type Fury — Premium Typing Experience

![Type Fury Logo](assets/bg_city.png)

**Type Fury** is a high-performance, browser-based typing game designed with a premium "Cyberpunk/Neon" aesthetic. It features multiple game modes that challenge speed, accuracy, and rhythm through dynamic environments and procedural difficulty.

## 🚀 Live Demo
Experience the game at [https://typefury.online/](https://typefury.online/)

---

## 🎮 Game Modes

### 🏎 Racing Mode
A competitive side-scrolling typing race against AI opponents.
- **Goal**: Reach the finish line first by typing common words accurately.
- **Features**: Real-time position tracking, dynamic particle effects, and scaling AI difficulty.

### ⚡ Neon Leak (Artillery)
A rhythm-based flow stabilization mode.
- **Goal**: Stabilize the system by typing words as they emerge from the "Neon Core."
- **Features**: Infinite level progression, adaptive difficulty, and a "Flow State" multiplier.

### 🌌 Galaxy Ops
A high-stakes interstellar typing siege.
- **Goal**: Defend your station from incoming enemy ships and viruses.
- **Features**: Sprite-based graphics, wave-based progression, and letter-by-letter typing feedback.

## 🛠 Tech Stack
- **Core**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Modern CSS3 (Grid, Flexbox, Custom Design System)
- **Audio**: Web Audio API (Procedural Sound FX & Rhythm Tracks)
- **Deployment**: GitHub Pages with automated CI/CD.
